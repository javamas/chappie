# chappie
Bot of development support

## Install
```Shell
git clone https://github.com/javamas/chappie.git
cd chappie
npm install
```
## Variables
```Shell
# require
export token=${SLACK_TOKEN}

# option(use message queue)
export MQ_HOST=${HOST}
export MQ_USER=${USER}
export MQ_PASSWORD=${PASSWORD}
```

## Run
```Shell
node bot.js
```
